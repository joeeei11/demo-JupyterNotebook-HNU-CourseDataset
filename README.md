# demo-JupyterNotebook-HNU-CourseDataset

**湖南科技大学课程作业数据集与分析笔记，包含 TEP 等标准工业过程数据集。**

## 功能特性

- TEP（田纳西-伊斯曼过程）标准数据集（d00–d21）
- 数据加载与预处理示例
- Jupyter Notebook 格式，便于交互式探索
- 支持故障检测与分类实验

## 快速开始

### 环境要求

- Python >= 3.8
- Jupyter Notebook / JupyterLab
- numpy, pandas, matplotlib

### 安装步骤

```bash
git clone https://github.com/joeeei11/demo-JupyterNotebook-HNU-CourseDataset.git
cd demo-JupyterNotebook-HNU-CourseDataset
pip install jupyter numpy pandas matplotlib scikit-learn
```

### 基础用法

```bash
jupyter notebook
```
